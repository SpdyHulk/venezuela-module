# venezuela-module
Package venezuela-module es un Go module para los: Estados, ciudades, municipios y parroquias.

## Dependencias

- [Gorm](https://github.com/jinzhu/gorm) - ORM
- [Maracon](https://github.com/go-macaron/macaron) - Api Restful
